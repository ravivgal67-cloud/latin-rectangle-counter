What is a Latin Rectangle?

A Latin rectangle is an r√ón array where each row is a permutation of {1, 2, ..., n} and no column contains duplicate values.

Example (2√ó3): 
1  2  3
2  3  1
‚ú®

Normalized Form

A normalized Latin rectangle has its first row as the identity permutation: [1, 2, 3, ..., n].

This reduces the search space while maintaining all essential properties!

Positive vs Negative

Each rectangle has a sign based on the parity of its row permutations:

  • Positive (+1): Even number of inversions
  • Negative (-1): Odd number of inversions

üîó Latin Squares (r=n)

When r=n, Latin rectangles become Latin squares. The Alon-Tarsi conjecture specifically concerns n√ón Latin squares and states that for even n, the number of even Latin squares differs from the number of odd Latin squares.

üîó Alon-Tarsi Number

An even integer n is called an Alon-Tarsi number if every (n-1)-regular bipartite graph is n-edge-colorable. Equivalently, n is an Alon-Tarsi number if the number of even n√ón Latin squares differs from the number of odd ones.

üîó Computational Exploration

While proven for some small cases, the general conjecture remains open. Computational tools like this help gather evidence, discover patterns, and test the conjecture for specific dimensions.
